### 文件解释

| 文件名           | 作用           |
| ---------------- | -------------- |
| requirements.txt | 所需第三方库   |
| data.json        | 成语数据文件   |
| gui.ui           | qt界面文件     |
| gui.py           | 转换后界面文件 |
| mian.py          | 程序入口       |

### 使用说明

选中文字后拖入框内即可，正确答案为最后三个字，如需随机可在main.py文件内116行代码解除注释。

### 注意事项

开发环境为python3.11
